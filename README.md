# 📊 Sales Performance Dashboard

## 📌 Project Overview

This project analyzes retail sales data to evaluate business performance, identify profitability issues, and support data-driven decision-making.

The analysis was performed using **Python (Pandas)** for data cleaning and exploratory analysis, followed by **Power BI** to build an interactive dashboard that summarizes key business metrics and insights.

---

## 🎯 Business Questions

This project aims to answer the following questions:

- Which product categories generate the highest profit?
- Which sub-categories are reducing overall profitability?
- Which regions perform best and worst?
- How do sales and profit change over time?
- Which areas require further investigation?

## 🔍 Key Insights

- Technology achieved the highest profit margin.
- Furniture generated high sales but the lowest profit margin.
- Tables and Bookcases were the primary loss-making sub-categories.
- The Central region recorded the weakest profitability.
- Analysis suggests that discount alone does not explain the losses, indicating that additional factors may be affecting profitability.
- Shipping Cost data would help better understand the losses in the Central region.

---

## 💼 Business Recommendations

- Review the pricing strategy for Furniture products.
- Re-evaluate discounts applied to low-profit products.
- Investigate operational costs in the Central region.
- Analyze shipping costs in future studies to identify the root cause of losses.
- Monitor the performance of loss-making sub-categories.
